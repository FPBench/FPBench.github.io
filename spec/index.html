FPBench 1.1

FPBench 1.1 adds support for mixed-precision computations and clarifies the semantics of rounded operations.

  • FPCore 1.1: syntax and semantics for benchmarks.
  • Examples 1.1: FPCore example inputs
  • Metadata 1.1: descriptive properties for benchmarks
  • Measures 1.1: standard error measurements
  • Notes for Implementors: how to use the standards

Changes:

  • Support for multiple and mixed-precision computations
  • Precision can be specified per-operation using the rounding context
  • Semantics of numerical operations clarified as rounded real operations
  • New number formats like exact rationals and hexadecimal floating-point
  • New metadata properties :round and :spec
