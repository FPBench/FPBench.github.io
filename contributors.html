The FPBench benchmarks are drawn from user contributions.

Contribute new benchmarks via Github.

File an issue on Github to contribute benchmarks. If your benchmarks have been used in published research, please include the reference in your bug report. Any other metadata you have about the benchmarks is appreciated: where the code comes from, what it does, and what sorts of inputs it is run on.
